变量的几个相关函数

  • isset：判断变量是否存在，判断变量是否定义了，同时它还可以判断变量的值是否为null
  • empty：判断变量是否为空，为空的值有:"",0,"0","0.0",false,[]
  • unset：删除变量
echo $a;//会报错

//可以改写成
if (isset($a)) {
    echo $a;
}

$arr = [];
for($i=0;$i<100;$i++){
    echo $arr[$i];
}

//可以改写成
$arr = [];
if(!empty($arr)){
    for ($i = 0; $i < 100; $i++) {
        echo $arr[$i];
    }
}

$a = 100;
echo $a;
unset($a);
echo $a;
